### java i++ 与 ++i 的区别

<!--more-->
java代码1
---

	{% highlight java %}
	public class Test {

	   public static void main(String[] args) {

		        int q = 0;
			q = q++;
			System.out.println(q);
	  }

	{% endhighlight %}

>输出结果 0
>
java代码2
---                                                    
                                                     
        {% highlight java %}                         
        public class Test {                          
                                                     
           public static void main(String[] args) {  
                                                     
                        int q = 0;                   
                        q = ++q;                     
                        System.out.println(q);       
          }                                          
                                                     
        {% endhighlight %}     

>输出结果 1
> 

出现这个现象的原因是由java的编译器确定的，即JVM
---
> 运行下面命令，查看反编译之后的代码
>
> javap -c Test
>
